## Usage
 Per esercizio CKY aprire il file __main__.py: 
	Scegliere la frase e il data( la grammatica da usare) tra quelle commentate e avviare il programma
	
 Per esercizio semantica aprire il file nltk_dothraki.py:
	Scegliere la frase tra quelle commentate e avviare il programma
	
## Grammatiche
	- dothraki_grammar_CF.txt: grammatica dothraki per esercizio CKY
	- L1_grammar_CF.txt: grammatica L1 per esercizio CKY
	- dothraki_gr_sem.fcfg: grammatica dothraki con semantica per esercizio nltk
